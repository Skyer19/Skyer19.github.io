Setting Git username for a single repository
  • Open Terminal.
  • Change the current working directory to the local repository where you want to configure the name that is associated with your Git commits.
  • Set a Git username: 
              git config user.name "Mona Lisa"
  • Confirm that you have set the Git username correctly: 
              git config user.name
Set the Github and Gitlab connection in a computer at the same time (Suitbale for the Mac OS)
  • Create the ssh key for different platform
    •       ssh-keygen -t rsa -f ~/.ssh/id_rsa_gitlab -C "user email"
      ssh-keygen -t rsa -f ~/.ssh/id_rsa_github -C "user email"
  • Modify the config file
    •       The file path:: ~/.ssh
    
          Host gitlab.doc.ic.ac.uk
      HostName gitlab.doc.ic.ac.uk 
      User User name or User email
      IdentityFile ~/.ssh/id_rsa_gitlab
      
      Host github.com
      HostName github.com
      User User name or User email
      IdentityFile ~/.ssh/id_rsa_github
  • Put the ssh public key to the Github/GitLab user setting

      ssh -T git@github.com
