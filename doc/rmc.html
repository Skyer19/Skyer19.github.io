Remote Connection

Set the Github and Gitlab connection in the computer at the same time

Suitbale for the Mac OS

Step 1: Create the ssh key for different platform

      ssh-keygen -t rsa -f ~/.ssh/id_rsa_gitlab -C "user email"
      ssh-keygen -t rsa -f ~/.ssh/id_rsa_github -C "user email"

Step 2: Modify the config file

      The file path:: ~/.ssh
    
      Host gitlab.doc.ic.ac.uk
      HostName gitlab.doc.ic.ac.uk 
      User User name or User email
      IdentityFile ~/.ssh/id_rsa_gitlab
      
      Host github.com
      HostName github.com
      User User name or User email
      IdentityFile ~/.ssh/id_rsa_github

Step 3: Put the ssh public key to the Github/GitLab user setting

ignore....

Step 4: Test the connection

      ssh -T git@gitlab.doc.ic.ac.uk
      ssh -T git@github.com
