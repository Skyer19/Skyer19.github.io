Linux Operation

Upload file via ssh 
      scp [path of file in my computer] [new path in the server]

      eg: scp [path of file in my computer] $USER@shell3.doc.ic.ac.uk:/homes/$USER/  // upload to the home path
下载压缩包和解压压缩包
下载压缩包 
      wget http://example.com/file
      wget -P path_of_directory http://example.com/file
解压压缩包 
      unzip file.zip
      unzip zip_file_path -d path_of_want_to_zip_directory
